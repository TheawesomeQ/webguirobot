# Web GUI Robot

## Controllers for robots on a self-hosted web server

The goal of this project is to run a webserver on a Raspberry Pi (3+) on a self-hosted wifi network, allowing the operator to connect to the network with their touchscreen device and control robot actions.

**Project components to be developed:**
* Web page
* Javascript for page and communication
* Python web server with Flask to host and Flask-socketIO to relay inputs

**Control elements:**
* Joystick
* Button
* Slider
